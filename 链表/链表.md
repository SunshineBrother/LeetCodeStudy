## 链表

我们上一章学习的，栈、队列，都是静态线性数据结构，底层依托于静态数组，依靠resize解决固定容量问题。

但是链表是真正的动态数据结构，也是最简单的动态数据结构，我们学习链表能够更加深入的理解指针，同时也能更加深入的理解递归函数。

链表的数组存储在节点(Node)中，节点通常有两部分组成，当节点数据为空的时候，就是最后一个节点
```
class Node {
    E e;        //节点数据
    Node next;  //指向下一个节点
}
```
